Windows Server

Windows Server: Windows kurulumlarının çoğu, masaüstü ve dizüstü bilgisayarlarda masaüstü kurulumları olarak gerçekleştirilir. Esas olarak Windows adı verilen veri merkezlerinde kullanılan başka bir Windows sürümü vardır. Sunucu. Bu, Windows Server 2003 ile başlayan bir Microsoftft ürünleri ailesidir. Windows Server birçok farklı hizmeti barındırır ve bir şirket içinde farklı rolleri yerine getirebilir Bunlar,

Windows Server'ın sağladığı hizmetlerin bazı ana bilgisayarlarıdır:
  • Network Services: DNS, DHCP ,Terminal services , Network Controller ve Hyper-V network sanallaştırılması.
  • File Services: SMB , NFS ve DFS
  • Web Services: HTTP , HTTPS ve FTP
  • Manegement: Group policy ve Active Directory domain services control
