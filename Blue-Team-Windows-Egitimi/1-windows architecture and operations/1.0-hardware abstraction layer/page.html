Hardware Abstraction Layer


  • Hardware Abstraction Layer: Donanım ile işletim sisteminin çekirdeğinin iletişimini kuruyor.
  • Operating System Kernel:görevi ise işletim sistemi ile Donanım arasında ki iletişimi sağlıyor.
  • Other Kernel-Mode Drivers: diğer çekirdik sürücüleri donanımın ihityaç duyduğu sürücülerdir Örn : Anakart sürücüsü olabilir.
  • WiNDOWS APı:Uygulama proglama arayüzüdür . işletim sistemi çekirdiğinin yazılımla uyumlu çalışması için yapılmış bir modüldür.Bu yazılım sayesinde örnek olarak kullanıcı arayüzünü sağlar.Mouse ve keyboard gibi donanımların düzgün çalışmasını sağlar.
  • User-Mode Drivers:bütün donanımların sürücüleri çekirdek tarafında değil kullanıcı tarafında da olabilir bu uyumu sağlar.
  • File System Drivers:işletim sisteminin kurulması için formatlı bir Harddriver sağlıyor.
